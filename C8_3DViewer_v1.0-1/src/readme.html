3D Viewer

О программе

    Программа для просмотра 3D моделей в каркасном виде. Каркасная модель - модель объекта в трёхмерной графике, представляющая собой совокупность вершин и рёбер, которая определяет форму отображаемого многогранного объекта в трехмерном пространстве. Модели необходимо загружать из файлов формата .obj. В программе имеется возможность просматривать модель на экране, а также вращать, масштабировать и перемещать.

Формат представления описаний трехмерных объектов .obj

    Формат файла .obj - это простой формат данных, который представляет только трехмерную геометрию, а именно положение каждой вершины, положение UV координат текстуры каждой вершины, нормали вершин и грани, которые определяют каждый многоугольник как список вершин и вершин текстуры. Координаты obj не имеют единиц измерения, но файлы obj могут содержать информацию о масштабе в удобочитаемой строке комментариев. В данном проекте реализована поддержка только списка вершин и поверхностей.

Общая информация

Кнопки

  • 'Open file' - Открыть файл .obj для просмотра. После открытия рядом отображается абсолютный путь до файла.
  • 'Screenshot' - Сохранить изображение модели в формате .jpeg или .bmp.
  • 'Create gif' - Запустит 5 секундный таймер записи области отображения модели, после чего позволит сохранить полученную анимацию в формате .gif.
  • 'Restore default settings' - Сбросить параметры отображения (цвета, отображение вершин, толщины и тип линий) к значениям по-умолчанию.

Блоки

  • 'Projection type' - Позволяет задать тип проекции отображения модели: параллельная (ортогональная) или центральная (перспектива).
  • 'Сolor' - Позволяет задать цвета фона (Background) граней (Edges) и вершин (Vertices).
  • 'Vertices type' - Позволяет изменить тип отображения вершин: none - не отображать, Square - отображать квадратами, Round - отображать кругами.
  • 'Vertices size' - Позволяет задать размер отображаемых вершин в пикселях.
  • 'Edges type' - Позволяет задать тип отрисовки граней: Solid - сплошные линии, Dashed - пунктирные линии.
  • 'Edges thikness' - Позволяет задать толщину граней.
  • 'Rotate' - Позволяет вращать модель вокруг своей соответствующей оси на указанное число градусов (отрицательное число соответствует повороту по часовой стрелке, положительное - против).
  • 'Move' - Позволяет перемещать модель вдоль соответствующей оси (X - слева-направо, Y - снизу-вверх, Z - от наблюдателя к наблюдателю).
  • 'Scale' - Позволяет изменять размер модели в указанное количество раз.

Установка

Чтобы установить программу, необходимо использовать команду "make install".

Запуск

Чтобы начать использовать программу, необходимо использовать команду "make open".

Удаление

Для удаления программы вы можете использовать команду "make uninstall".
