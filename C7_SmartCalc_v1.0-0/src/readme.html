SmartCalc

О программе

    Расширенная версия обычного калькулятора, который можно найти в стандартных приложениях каждой операционной системы. Помимо базовых арифметических операций, как плюс-минус и умножить-поделить, калькулятор имеет возможность вычисления арифметических выражений с учетом приоритетов, а так же некоторых математических функций (синус, косинус, логарифм и т.д.). Помимо вычисления выражений калькулятор так же поддерживает использование переменной x и построение графика соответствующей функции.

Арифметические операторы

  • '( )'  Скобки
  • '+'    Сложение или унарный плюс
  • '-'    Вычитание или унарный минус
  • '*'    Умножение
  • '/'    Деление
  • '^'    Возведение в степень
  • 'mod'  Остаток от деления

Функции

  • 'sin'  Вычисляет синус
  • 'asin' Вычисляет арксинус
  • 'cos'  Вычисляет косинус
  • 'acos' Вычисляет арккосинус
  • 'tan'  Вычисляет тангенс
  • 'atan' Вычисляет арктангенс
  • 'ln'   Вычисляет натуральный логарифм
  • 'log'  Вычисляет десятичный логарифм
  • 'sqrt' Вычисляет квадратный корень

Дополнительно

  • 'x'    Вводит X для дальнейшей подстановки или построения графика (для подстановки необходимо ввести значение в поле слева)
  • 'С'    Очистить поле ввода
  • '<--'  Удалить последний введенный оператор, цифру или функцию
  • 'Rad'  Использовать для расчёта радианную меру
  • 'Deg'  Использовать для расчёта градусную меру

Построение графика

    Для построения графика функции необходимо задать функцию в главном окне ввода, указать границы областей определения и значений, после чего нажать кнопку 'GRAPH'

Поля для ввода областей значений и определения

  • 'min X' Левая граница области определения (должна быть меньше правой)
  • 'max X' Правая граница области определения (должна быть больше левой)
  • 'min Y' Нижняя граница области значений (должна быть меньше верхней)
  • 'max Y' Верхняя граница области значений (должна быть больше нижней)

Кредитный калькулятор

    Кредитный калькулятор предназначен для расчёта ежемесячных выплат по кредиту, переплаты и общей суммы выплат за период.

Поля для ввода

  • 'Total credit amount' Общая сумма кредита (не может быть меньше 0.01 рубля)
  • 'Credit term'         Срок кредита (не может быть меньше 1 месяца, также обязательно должен быть целым числом)
  • 'Interest rate'       Процентная ставка по кредиту (не может быть меньше 0.01% и больше 999%)
  • 'Payment type'        Способ расчёта ежемесячных платежей по кредиту - Аннуитетный или Дифференцированный

Поля вывода

  • 'Monthly payment'     Общая сумма кредита (не может быть меньше 0.01 рубля)
  • 'Overpayment'         Срок кредита (не может быть меньше 1 месяца, также обязательно должен быть целым числом)
  • 'Total payment'       Процентная ставка по кредиту (не может быть меньше 0.01%)

Установка

    Чтобы установить калькулятор, необходимо использовать команду "make install".

Запуск

    Чтобы начать использовать калькулятор, необходимо использовать команду "make open".

Удаление

    Для удаления программы вы можете использовать команду "make uninstall".
